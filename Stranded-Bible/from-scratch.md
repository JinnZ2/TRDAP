🌍 PART 1: THE PHILOSOPHY - Why Scrap?

You've asked the most important question: What happens when the supply chains stop?

When the ships don't come, when the factories close, when Amazon is just a rainforest—people will still need hope, communication, and survival tools. The difference between life and death will be whether they can make what they need from what they have.

This isn't just about recycling. This is about radical self-sufficiency. Every component in The Ark should be manufacturable from:

· Electronic waste (old TVs, radios, computers)
· Household junk (cans, bottles, wires)
· Natural materials (wood, plant extracts, metals from soil)
· Scrap yards and landfills (the world's richest resource)

🔧 PART 2: COMPONENT-BY-COMPONENT - BUILDING FROM SCRATCH

A. CONDUCTORS - Making Wires and Traces

From Scrap Copper

```
SOURCE MATERIALS:
• Old electrical wiring (houses, appliances)
• Motor windings (broken fans, motors)
• Transformer coils (TVs, power supplies)
• Copper pipe (plumbing scraps)
• ROM boards from old computers

PROCESS:
1. Strip insulation (burn carefully or scrape)
2. Anneal: Heat to cherry red, quench in water
3. Draw wire: Pull through progressively smaller holes
   - Drill holes in hardened steel plates
   - Start thick, work down to desired gauge
4. For flat traces: Hammer annealed wire into thin strips
```

Alternative Conductors

```
ALUMINUM:
• From cans, gutters, old windows
• Clean with steel wool
• Cannot solder well—must crimp or use mechanical connections

BRASS/BRONZE:
• From old door hardware, plumbing fixtures
• File or sand to shape
• Good for contacts, switches

GRAPHITE:
• From pencils (artificial) or rocks (natural)
• Crush and mix with binder for conductive paint
• Draw traces on paper or wood [citation:6]

STEEL:
• From coat hangers, tin cans
• High resistance—use only for structural/mechanical
• Spot weld, don't solder

LIQUID METAL:
• Gallium from old LEDs, semiconductors
• Indium from touchscreens, some solder
• Mix for EGaIn (eutectic gallium-indium) [citation:1][citation:5]
```

Conductive Thread

```
MATERIALS:
• Stainless steel from old kitchenware (file into thin strips, then draw)
• Copper from wires (same process)
• Silver from old coins, jewelry, industrial scrap

PROCESS:
1. Create extremely fine wire by drawing
2. Twist multiple strands together
3. Use as thread for fabric circuits [citation:7]
```

C. RESISTORS

From Graphite

```
SOURCE:
• Pencil leads (all grades)
• Carbon from old batteries
• Crushed coal or charcoal

METHOD 1 - Drawn:
1. Mix graphite powder with binder (glue, shellac)
2. Paint line on board
3. Longer line = higher resistance
4. Thicker line = lower resistance
5. Test with multimeter, scrape to adjust [citation:6]

METHOD 2 - Wirewound:
1. Find high-resistance wire (old nichrome heater elements, toaster wires)
2. Wind around non-conductive core
3. More turns = higher resistance
4. Tap at different points for different values

METHOD 3 - Composition:
1. Mix graphite powder with clay or plaster
2. Form into small cylinders
3. Fire in kiln or oven
4. Attach wire ends
```

From Metal Film

```
1. Find old metal film resistors (every device has them)
2. Desolder carefully
3. Measure and sort by value
4. Reuse directly

Or:
1. Sputter metal onto glass (advanced)
2. Scratch spiral path
3. Value determined by spiral length
```

Natural Resistive Materials

```
• Salt water (variable)
• Wet wood (unstable)
• Metal oxides (from rust)
• Semiconductor crystals (galena, silicon from old electronics)
```

D. CAPACITORS

Paper and Foil

```
MATERIALS:
• Aluminum foil (kitchen foil, capacitor foil from old electronics)
• Wax paper, parchment, or plastic wrap
• Old wax candles for impregnation

PROCESS:
1. Cut two strips of foil (plates)
2. Cut three strips of paper (insulators)
3. Stack: paper-foil-paper-foil-paper
4. Roll tightly
5. Impregnate with wax (heat wax, dip)
6. Attach wire leads
7. Encapsulate in wax or epoxy

CAPACITANCE determined by:
• Area of foil (larger = more)
• Distance between plates (thinner paper = more)
• Dielectric constant (wax paper ~3.5)
```

Air Variable

```
FROM OLD RADIOS:
• Remove tuning capacitor
• Clean contacts
• Reuse directly

FROM SCRATCH:
1. Cut aluminum plates (from cans)
2. Mount on shaft with insulating spacers
3. Stationary plates interleave with moving plates
4. More overlap = higher capacitance
```

Electrolytic (Re-forming)

```
FROM OLD DEVICES:
• Even old electrolytics can be re-formed
• Apply voltage through resistor (1k per volt)
• Leave for hours to rebuild oxide layer
• Test before using

FROM SCRATCH - Advanced:
1. Etch aluminum foil (salt water, electric current)
2. Anodize in boric acid solution
3. Roll with paper soaked in electrolyte
4. Requires careful chemistry but possible
```

Natural Dielectrics

```
• Mica (from old heaters, or mined)
• Glass
• Ceramic (from old insulators)
• Oil
• Air
```

E. INDUCTORS and TRANSFORMERS

Air Core Coils

```
MATERIALS:
• Copper wire (reclaimed)
• Plastic or cardboard form
• Old solenoid from broken appliance

PROCESS:
1. Wind wire around form
2. More turns = higher inductance
3. Closer turns = higher inductance
4. Larger diameter = higher inductance

CALCULATION:
• For single layer: L (uH) = (diameter² × turns²) / (18d + 40l)
• Where d = diameter in inches, l = length in inches
```

Iron Core

```
CORES FROM:
• Old transformers (salvage)
• Ferrite from computer cables (snap-on beads)
• Iron powder from old inductors
• Laminated steel from motor stators

PROCESS:
1. Salvage core
2. Wind primary (if needed)
3. Wind secondary (for desired ratio)
4. Insulate between layers with paper or tape
```

Ferrite from Scratch (Advanced)

```
1. Mix iron oxide (rust) with nickel/zinc oxides
2. Add binder, form desired shape
3. Fire at high temperature (kiln)
4. Test magnetic properties
```

F. SEMICONDUCTORS (The Hard Part)

Salvage First

```
EVERY OLD DEVICE CONTAINS:
• Transistors (hundreds in a TV)
• Diodes (power supplies, rectifiers)
• ICs (processors, memory, logic)
• LEDs (indicators, displays)

DESOLDERING:
1. Heat with iron or hot air
2. Use solder sucker or braid
3. Work quickly to avoid damage
4. Test with multimeter (diode mode)
```

Point-Contact Diode

```
MATERIALS:
• Galena crystal (lead sulfide) - can be found in some rocks
• Fine wire (phosphor bronze, steel)
• Spring to hold contact

PROCESS:
1. Mount crystal in holder
2. Position fine wire against crystal
3. Adjust pressure to find "sweet spot"
4. Used in crystal radios—works as detector

Also try:
• Silicon from broken chips (scratch surface)
• Rusty razor blade and pencil lead (foxhole radio)
```

Early Transistors (Advanced)

```
POINT-CONTACT:
• Two fine wires touching germanium or silicon
• Extremely finicky, low gain
• Historically accurate but not practical

JUNCTION:
• Requires crystal growing and doping
• Beyond home workshop without serious chemistry
• Focus on salvage instead
```

G. BATTERIES

Voltaic Pile (Zinc-Copper)

```
MATERIALS:
• Zinc (from galvanized nails, old battery cases)
• Copper (wire, pipe, pennies pre-1982)
• Electrolyte: salt water, vinegar, lemon juice
• Cardboard or paper separators

PROCESS:
1. Cut zinc and copper into discs
2. Soak paper in electrolyte
3. Stack: copper-paper-zinc-paper-copper...
4. Each cell ~0.8-1.0V
5. Stack for desired voltage
```

Lemon/Potato Battery

```
1. Insert zinc nail and copper wire into fruit
2. Each cell ~0.9V
3. Series for higher voltage
4. Low current—good for LED, not radio
```

Rechargeable Cells (Salvage)

```
• Old laptop batteries (18650 cells)
• Power tool packs
• Emergency light batteries
• Test before use—discard if voltage <2.5V

REFURBISH:
1. Remove from pack
2. Charge slowly
3. Test capacity
4. Re-wrap in heat shrink
```

Lead-Acid from Scrap

```
PLATES:
• Cut from old battery
• Or cast from lead (wheel weights, pipe)

ELECTROLYTE:
• Sulfuric acid (from old batteries, drain cleaner)
• Dilute with distilled water (1.265 specific gravity)

HOUSING:
• Plastic container
• Wood box lined with tar
• Glass jar

Each cell ~2.1V
```

Aluminum-Air (Emergency)

```
1. Aluminum can (clean, expose fresh metal)
2. Carbon rod (from old battery)
3. Salt water electrolyte
4. Works for hours, not rechargeable
```

H. SOLAR CELLS

From Garden Lights

```
CHEAPEST SOURCE:
• Dollar store solar garden lights
• Each contains small solar cell (0.5-2W)
• Also has NiMH battery and sometimes useful LED

PROCESS:
1. Break open plastic housing
2. Carefully remove solar cell (fragile!)
3. Test voltage in sun (typically 2-4V)
4. Connect multiple in series for higher voltage [citation:8]
```

From Broken Panels

```
• Check solar installers for damaged panels
• Even broken panels have functional cells
• Cut out good cells
• Re-tab with salvaged wire

PROCESS:
1. Cut panel open (carefully)
2. Identify working cells (test each)
3. Remove from backing
4. Re-solder tabbing wire
5. Encapsulate in epoxy or between glass
```

From Scratch (Advanced)

```
• Copper oxide cell: Heat copper sheet until black, cool slowly
• Produces tiny current—demonstration only
• TiO2 dye-sensitized cells possible with chemistry
```

L. POWER MANAGEMENT

Voltage Regulation

Zener Diodes:
• Salvage from old power supplies
• Test voltage with variable supply
• Use as reference

Linear Regulators:
• 7805, 1117 common in many devices
• Salvage, test, reuse
• Add capacitors for stability

From Scratch - Zener from Transistor:
• Use base-emitter junction (6-7V breakdown)
• Connect base to collector
• Use as crude regulator

Boost Converters

Joule Thief:

1. Ferrite core (from old CFL bulb)
2. Wind two windings (10+10 turns)
3. Transistor (any NPN)
4. Resistor (1k)
5. LED will light from dead battery

From Scrap:
• Boost controller ICs in many devices
• Salvage, identify, reuse
• Inductors from old power supplies

Battery Charging

Simple Resistor Limiting:

1. Calculate R = (Vsupply - Vbat) / Icharge
2. Use power resistor
3. Monitor voltage, disconnect when full

Solar Direct:
• Panel voltage slightly above battery
• Diode to prevent backflow
• Works for NiMH, careful with Li-ion

Phase 2: Sorting and Testing

Organization System

```
By type:
• Resistors (sorted by color code)
• Capacitors (by value/voltage)
• Transistors (by part number)
• Diodes (by part number)
• ICs (by part number)
• Hardware (screws, brackets)
• Wire (by gauge)
• Motors (by size/voltage)
```

Testing Everything

```
Resistors:
• Measure with multimeter
• Sort by value
• Note tolerance

Capacitors:
• Check for shorts
• Measure capacitance (if meter has function)
• Check ESR (if possible)
• Look for bulging/leaking

Diodes:
• Diode mode: should read one way (~0.6V)
• Open the other way
• Test LEDs (lights up dimly)

Transistors:
• Diode test between base-emitter, base-collector
• Should read like diodes
• No shorts between collector-emitter
• Search part number for pinout

ICs:
• Visual inspection (cracks, burns)
• Can't fully test without circuit
• Set aside for later experimentation
```

Phase 3: Building the Minimal Ark

Version 1: The Crystal Radio Core

```
COMPONENTS:
• Coil (wire on cardboard tube)
• Variable capacitor (from old radio)
• Diode (germanium from old radio, or 1N4148/1N34)
• High-impedance earphone (crystal earphone, or piezoelectric)
• Antenna (long wire)
• Ground (pipe or rod)

BUILD:
1. Wind 80 turns of wire on 1" tube
2. Tap at 20 turns (for antenna connection)
3. Connect capacitor across coil
4. Connect diode from top of coil to earphone
5. Earphone other end to ground
6. Antenna to tap

RECEIVES: AM broadcasts - weather, news, information
NO BATTERIES NEEDED - powered by radio waves
```

Version 2: Add Audio Amplification

```
SALVAGE:
• LM386 audio amp IC (common)
• Or build from transistors (2-3 NPN)
• Speaker from old radio

CIRCUIT:
• Crystal radio output → volume pot → amp input
• Amp output → speaker
• Power from salvaged battery

HEARS: Much louder, can fill small area
```

Version 3: Add Microcontroller

Version 4: Add Power System

```

• Solar panel from garden lights (multiple)

• Voltage regulator from scrap board

BUILD:

```

Version 5: Add Mesh Communication

```
SALVAGE:
```

📻 MODULE 1: THE CRYSTAL RADIO - Your First Connection

No batteries. No chips. No power. Just wire, metal, and magic.

---

1.1 Why This Matters

The crystal radio is the original off-grid communication device. It:

· Requires zero power (runs on radio waves themselves)
· Can be built from almost anything
· Receives weather alerts, news, and information from stations hundreds of miles away
· Teaches you the fundamentals of radio that everything else builds on
· Works when everything else fails

In a survival situation, information is survival. Knowing about approaching weather, rescue efforts, or where help is available could save your life.

---

1.2 The Theory (Simple Version)

```
Radio waves are everywhere—invisible energy traveling through the air.

Your radio does three things:
1. CAPTURE the waves (antenna)
2. SELECT one station (tuned circuit)
3. CONVERT to sound (detector + earphone)

That's it. The energy to make sound comes FROM the radio waves themselves.
```

1.4 Building Each Component from Scratch

A. Making Your Own Wire

If you can't find magnet wire, make it:

```
MATERIALS NEEDED:
• Any copper wire (thick)
• Sandpaper or file
• Hammer
• Drill (optional)

PROCESS:
1. Start with thick copper wire (from house wiring)
2. Hammer it flat (makes it thinner)
3. File the edges smooth
4. Twist to make round again
5. Repeat until desired thinness
6. For very fine wire: clamp one end in drill, other in vise
7. Spin drill while pulling gently - stretches wire thinner

NOTE: This takes time. A few feet of fine wire might take hours.
But in a survival situation, you have time.
```

B. Making a Variable Capacitor

A variable capacitor lets you tune different stations. If you can't find one:

```
MATERIALS:
• 2 sheets of aluminum (from cans)
• Plastic from bottle
• Bolt and nuts
• Wire scraps

PROCESS:
1. Cut two strips of aluminum: 1" x 4"
2. Cut plastic slightly larger for insulation
3. Sandwich: aluminum - plastic - aluminum
4. Bolt through one end (insulate from second plate)
5. Connect wire to each plate
6. Bending plates closer = more capacitance
7. Spreading apart = less capacitance

This is crude but works. You'll "tune" by physically bending.
```

Better version (air variable):

```
1. Cut multiple aluminum plates (from cans)
2. Punch holes in all
3. Alternate: fixed plates connected together, moving plates connected
4. Mount on bolt with plastic washers between
5. Rotating shaft changes overlap = tuning

Think of it as a "sandwich" of plates that slide between each other.
```

C. Making a Diode

The hardest part. Germanium diodes work best, but you can make:

Option 1: Razor Blade Diode (Foxhole Radio)

```
MATERIALS:
• Rusty razor blade (or any steel with blue oxide)
• Safety pin or fine wire
• Pencil lead (graphite)

PROCESS:
1. Heat razor blade until blue oxide forms (not red hot)
2. Mount blade on board
3. Connect safety pin as "cat's whisker"
4. Press lightly on blade
5. Move until you find a "sweet spot" that detects

This actually works. WW2 soldiers built these in foxholes.
```

Option 2: Crystal Diode

```
MATERIALS:
• Galena crystal (lead sulfide - found in some rocks)
• Fine wire (phosphor bronze or steel)
• Spring or weight

PROCESS:
1. Mount crystal in holder (melt solder around it)
2. Position fine wire against crystal
3. Adjust pressure to find detection point
4. Once found, lock in place

Galena was the original "crystal" in crystal radios.
```

Option 3: Salvage First
But honestly? Find a real diode. They're everywhere. 1N4148 (silicon) works, though not as sensitive as germanium. Any electronics board will have dozens.

D. Making a High-Impedance Earphone

Regular headphones (32 ohms) won't work well. You need 2000+ ohms.

Option 1: Piezoelectric Element

```
MATERIALS:
• Piezo buzzer (from smoke detector, greeting card, microwave)
• Paper cup or cone

PROCESS:
1. Remove piezo element from buzzer
2. Solder wires to it
3. Glue to paper cone for better sound
4. Listen directly - no amplifier needed

Works great! Piezo elements are very high impedance.
```

Option 2: Telephone Earpiece

```
MATERIALS:
• Old telephone handset
• Cut cord, find two wires
• Connect directly
```

Option 3: Speaker from Headphone

```
MATERIALS:
• Old headphones
• Tiny speaker inside

PROBLEM: Too low impedance
SOLUTION: Add transformer (see below)
```

Option 4: Winding Your Own

```
MATERIALS:
• Fine wire (from motor)
• Magnet (from speaker)
• Thin membrane (paper, plastic)
• Tube former

PROCESS:
1. Wind 1000+ turns of fine wire on small tube
2. Mount near magnet with gap
3. Attach paper cone to coil
4. Connect to radio

Tedious but possible. This is how speakers work.
```

E. Making an Impedance Matching Transformer

If you only have low-impedance headphones (32 ohms):

```
MATERIALS:
• Ferrite core (from CFL bulb, old transformer)
• Fine wire (primary)
• Thicker wire (secondary)

RATIO: About 10:1 turns ratio (100 turns primary : 10 turns secondary)

PROCESS:
1. Wind 100 turns fine wire on core (primary)
2. Wind 10 turns thicker wire over it (secondary)
3. Connect primary to radio
4. Connect secondary to low-impedance headphones

This "transforms" the impedance to match.
```

1.5 Building the Basic Crystal Radio

Schematic (Simple Version)

```
Antenna ─────┬──── Coil (tapped) ────┬──── Diode ──── Earphone ──── Ground
             │                       │
             └── Variable Capacitor ──┘
```

Step-by-Step Assembly

Step 1: Wind the Coil

```
1. Find a form: cardboard tube (1-2" diameter)
2. Scavenge magnet wire (28-32 gauge is ideal)
3. Wind 80 turns, close together
4. Make a "tap" at 20 turns (scrape insulation, solder a wire)
5. Secure with tape or wax
```

Step 2: Connect the Variable Capacitor

```
1. Identify the two sections of your salvaged capacitor
2. Connect one side to top of coil (where antenna connects)
3. Connect other side to bottom of coil (ground side)
4. The capacitor and coil together form the "tuned circuit"
```

Step 3: Add the Diode

```
1. Identify diode polarity (banded end = cathode)
2. Connect diode anode to coil tap (20 turns point)
3. Connect diode cathode to one earphone wire
```

Step 4: Complete the Circuit

```
1. Connect other earphone wire to ground
2. Connect antenna wire to top of coil
3. Connect bottom of coil to ground
4. Add a ground connection (metal pipe, rod in earth)
```

Step 5: Test

```
1. String up antenna as high and long as possible
2. Ensure good ground connection
3. Adjust variable capacitor slowly
4. Listen carefully - you should hear stations

If no sound:
• Check all connections
• Try different tap point on coil
• Try reversing diode
• Add more antenna
• Try at night (better reception)
```

---

1.6 Improving Your Crystal Radio

Add a Second Diode (Full Wave)

```
Two diodes can double the volume:

Antenna ── Coil ──┬── D1 ── Earphone (+) ──┬── Ground
                  │                         │
                  └── D2 (reversed) ────────┘

D1 and D2 in opposite directions give both halves of wave.
```

Add an Amplifier (If You Have Power)

```
Any old transistor can amplify:

Crystal radio out ──┬── 0.1uF cap ── Transistor base
                    │
                    └── 1M resistor ── +V
                    
Transistor collector ── earphone ── +V
Transistor emitter ── ground

Now you need a battery, but volume increases dramatically.
```

Better Antenna

```
• Longer is better (50-100 feet ideal)
• Higher is better (up in trees)
• Insulated from ground
• "Inverted L" shape works well
```

Better Ground

```
• Metal pipe driven into earth
• Copper plate buried
• Connect to cold water pipe (if metal)
• Multiple ground rods connected
```

The crystal radio needs no power, but everything else—amplifiers, microcontrollers, lights, communication—needs electricity. Solar power gives you:

Think of it as:

· Sunlight = energy to run the pump
· Electricity = water flowing
· Battery = water tank
· Your device = sprinkler using stored water

2.4 Building Each Component from Scratch

A. Making Solar Cells from Garden Lights

This is the easiest source of small solar cells.

```
MATERIALS:
• Dead solar garden light (dollar store type)
• Small screwdriver
• Patience

PROCESS:
1. Remove stake and plastic cover
2. Look for small circuit board with solar cell on top
3. Solar cell is the brown/blue/black panel
4. Carefully pry cell from plastic (it's fragile!)
5. Note which wire is positive, which negative
6. Test in sun: should read 2-4V

Each garden light gives you one small cell (about 1-2 watts).
Collect many for more power.
```

B. Making Solar Cells from Broken Panels

If you can find a broken solar panel:

```
MATERIALS:
• Broken solar panel (check installers, recycling)
• Hammer
• Wire cutters
• Multimeter

PROCESS:
1. Panel is made of many individual cells
2. Even with cracks, many cells still work
3. Use multimeter to find working cells (0.5V each in sun)
4. Cut between cells (carefully)
5. Remove from backing
6. Test each cell individually

Connecting cells:
• Series = add voltage (cells in a row)
• Parallel = add current (cells side by side)
• Match cells roughly for best results
```

C. Making Solar Cells from Scratch (Copper Oxide)

This is experimental but possible:

```
MATERIALS:
• Copper sheet (from wire, pipe, roofing)
• Electric stove or hot plate
• Salt water
• Alligator clips

PROCESS:
1. Clean copper sheet thoroughly
2. Heat red hot for 30 minutes (forms black oxide)
3. Cool slowly (some black flakes off, leaving red oxide)
4. Assemble cell:
   - Copper oxide sheet
   - Salt water electrolyte
   - Another clean copper sheet
   - Separator (paper towel)
5. Connect wires: oxide side positive, clean side negative

OUTPUT: Tiny current (microamps), but it works!
This is how first solar cells were made.
```

D. Making a Battery from Scrap (Voltaic Pile)

If you can't find a rechargeable battery:

```
MATERIALS:
• Zinc (galvanized nails, old battery cases)
• Copper (wire, pennies pre-1982)
• Salt
• Water
• Paper or cardboard
• Container

PROCESS:
1. Cut zinc and copper into small squares
2. Soak paper in salt water
3. Stack: copper - paper - zinc - paper - copper...
4. Each "cell" = about 0.8-1.0V
5. Stack 4 cells for ~3-4V
6. Top and bottom with wire connections

This is a PRIMARY battery (not rechargeable).
But it works immediately.
```

E. Making a Charging Circuit (Simple)

The absolute simplest charger:

```
Solar Cell (+) ──── Diode ──── Battery (+)
Solar Cell (-) ───────────────── Battery (-)

DIODE: Any power diode (1N400x) prevents battery draining back at night

THAT'S IT. No overcharge protection. 
You must monitor battery voltage and disconnect when full.

For lithium batteries: DANGER - overcharge can cause fire.
Only use this for lead-acid or NiMH.
```

F. Making a Protection Circuit

For lithium batteries, protection is essential:

```
SALVAGE FIRST:
• Every lithium battery pack has protection circuit
• Carefully remove from old phone battery
• Note connections: B+, B-, P+, P-
• B = battery connections
• P = output connections

FROM SCRATCH? Nearly impossible.
Salvage is the way.
```

G. Making a Boost Converter (Joule Thief)

To get 5V from a 1.2V battery:

```
MATERIALS:
• Ferrite core (from CFL bulb)
• 28 AWG wire (from motor)
• NPN transistor (any, 2N2222 common)
• 1k resistor
• LED (optional)

PROCESS:
1. Wind 10 turns on core (collector winding)
2. Wind 10 more turns (base winding)
3. Connect:
   - Collector winding one end to +battery
   - Collector winding other end to transistor collector
   - Base winding one end to transistor base
   - Base winding other end to 1k resistor
   - Resistor other end to +battery
   - Transistor emitter to -battery
4. Output across collector winding

This oscillator steps up voltage - can light LED from "dead" battery.
```

2.5 Building a Complete Solar Charger

Option A: Simple Garden Light Charger

```
COMPONENTS:

3. Connect solar - to battery -
4. Connect protection circuit to battery (B+ and B-)
```

```
This is the easiest working solution.
```

Estimating Charge Time

```
Battery capacity in mAh / Solar current in mA = hours

Example:
2000mAh battery / 100mA solar = 20 hours full sun

But you rarely get full sun all day.
Realistically, count on 4-6 "equivalent sun hours" per day.
So 20 hours / 5 hours per day = 4 days to full charge.
```

Caring for Your Batteries

· Lithium-ion: Never over-discharge below 2.5V, never over-charge above 4.2V
· NiMH: More forgiving, but don't reverse charge
· Lead-acid: Heavy but durable, needs occasional full charge
· Test regularly with multimeter
· Disconnect if batteries get hot

3.2 The Theory (Simple Version)

```
A microcontroller is a tiny computer on one chip.

It has:
• PROCESSOR: Does calculations
• MEMORY: Stores program (flash) and data (RAM)
• PINS: Connect to buttons, LEDs, sensors
• TIMERS: Measure time, generate signals
• COMMUNICATION: Talk to other chips

You write a program on a computer, 
load it into the chip,
and the chip runs that program forever.
```

3.5 Building the Minimal Circuit

Absolute Minimum to Run a Chip

```
For ATMega328P (Arduino heart):

                    +5V
                     │
                     ├─── 10k ────┐
                     │             │
                    ┌┴┐           ┌┴┐
                    │ │ Reset      │ │
                    └┬┘           └┬┘
                     │             │
    16MHz ───┬── X1 ─┼─────────────┼── VCC
             │       │             │
           22pF      │            GND
             │       │
            GND     GND

Power: 5V to VCC, GND to GND
Crystal: Between XTAL1 and XTAL2
Capacitors: 18-22pF from each crystal pin to ground
Reset: Pulled high with 10k resistor to VCC

THAT'S IT. The chip will run.
```

Breadboard Prototype

```
1. Place chip in center of breadboard
2. Connect VCC and GND to power rails
3. Add 0.1uF capacitor near VCC (if you have one)
4. Install crystal and caps
5. Add reset resistor
6. Connect LED + resistor to a pin (test)
7. Power with 5V regulated supply
```

3.6 Powering Your Chip

Voltage Requirements

Chip Family Voltage Notes
ATMega (old) 4.5-5.5V Classic Arduino
ATMega (new) 1.8-5.5V Check datasheet
PIC 2.0-5.5V Varies by model
ARM 2.0-3.6V Most are 3.3V
ESP8266 3.0-3.6V 3.3V ONLY!
ESP32 3.0-3.6V 3.3V ONLY!

Making 5V from Scrap

```
FROM 3-4 CELLS (3.6-4.8V):
• Use boost converter (Joule thief or salvaged module)
• Some chips run directly at this voltage (check!)

FROM 6V SOLAR:
• Use 7805 linear regulator
• Connect: 6V in → 7805 in, 7805 out → 5V
• GND common
• Add capacitors (0.33uF in, 0.1uF out)

FROM 12V (car battery, crank):
• Use 7805 with heatsink
• 12V to 5V drops 7V as heat
• Only use for low current (<100mA)
```

Making 3.3V from Scrap

```
FROM 5V:
• Use AMS1117-3.3 regulator (common in boards)
• Connect 5V in, 3.3V out
• Add capacitors

FROM SCRAP:
• Find any 3.3V regulator on old board
• Salvage and reuse

FROM 2x AA BATTERIES (3V):
• May run directly on 3V
• Check chip minimum voltage
```

3.8 Minimal Code for The Ark

Basic Menu System (Arduino)

```cpp
// Minimal Ark Firmware
// Fits in 32KB ATMega328

#include <LiquidCrystal.h>

// Pin definitions
#define BTN_UP 2
#define BTN_DOWN 3
#define BTN_SEL 4
#define LED_PIN 13
#define SPEAKER 5

// LCD pins (if you have one)
LiquidCrystal lcd(7, 8, 9, 10, 11, 12);

// Menu structure
const char* menuItems[] = {
  "Prayers",
  "Survival",
  "Morse Code",
  "Journal",
  "Last Words",
  "Signal SOS"
};
const int menuCount = 6;
int menuPos = 0;

// Content (stored in PROGMEM to save RAM)
const char prayer1[] PROGMEM = "Psalm 23: The Lord is my shepherd...";
const char prayer2[] PROGMEM = "Our Father, who art in heaven...";
// Add more prayers...

void setup() {
  pinMode(BTN_UP, INPUT_PULLUP);
  pinMode(BTN_DOWN, INPUT_PULLUP);
  pinMode(BTN_SEL, INPUT_PULLUP);
  pinMode(LED_PIN, OUTPUT);
  pinMode(SPEAKER, OUTPUT);
  
  lcd.begin(16, 2);
  lcd.print("ARK v1.0");
  delay(2000);
  lcd.clear();
}

void loop() {
  // Display current menu
  lcd.setCursor(0, 0);
  lcd.print("> ");
  lcd.print(menuItems[menuPos]);
  lcd.setCursor(0, 1);
  if (menuPos < menuCount-1) {
    lcd.print("  ");
    lcd.print(menuItems[menuPos+1]);
  }
  
  // Handle buttons
  if (digitalRead(BTN_UP) == LOW) {
    menuPos--;
    if (menuPos < 0) menuPos = 0;
    delay(200);
  }
  
  if (digitalRead(BTN_DOWN) == LOW) {
    menuPos++;
    if (menuPos >= menuCount) menuPos = menuCount-1;
    delay(200);
  }
  
  if (digitalRead(BTN_SEL) == LOW) {
    selectMenu(menuPos);
    delay(300);
  }
}

void selectMenu(int item) {
  lcd.clear();
  switch(item) {
    case 0: showPrayers(); break;
    case 1: showSurvival(); break;
    case 2: showMorse(); break;
    case 3: journal(); break;
    case 4: lastWords(); break;
    case 5: signalSOS(); break;
  }
}

void showPrayers() {
  // Simple list of prayers
  lcd.print("Prayers");
  delay(2000);
  // Show first prayer
  lcd.clear();
  char buffer[17];
  strcpy_P(buffer, prayer1);
  lcd.print(buffer);
  delay(5000);
}

void signalSOS() {
  lcd.print("SENDING SOS");
  for(int i=0; i<3; i++) {
    // Three dots
    for(int d=0; d<3; d++) {
      digitalWrite(LED_PIN, HIGH);
      tone(SPEAKER, 800);
      delay(200);
      digitalWrite(LED_PIN, LOW);
      noTone(SPEAKER);
      delay(200);
    }
    delay(200);
    // Three dashes
    for(int d=0; d<3; d++) {
      digitalWrite(LED_PIN, HIGH);
      tone(SPEAKER, 800);
      delay(600);
      digitalWrite(LED_PIN, LOW);
      noTone(SPEAKER);
      delay(200);
    }
    delay(400);
  }
  lcd.clear();
}

// Other functions would be similar
void showSurvival() { lcd.print("Survival"); delay(2000); }
void showMorse() { lcd.print("Morse"); delay(2000); }
void journal() { lcd.print("Journal"); delay(2000); }
void lastWords() { lcd.print("Last Words"); delay(2000); }
```

3.10 Power-Saving Techniques

Sleep Modes

```cpp
// Put ATMega to sleep
#include <avr/sleep.h>

void goToSleep() {
  set_sleep_mode(SLEEP_MODE_PWR_DOWN);
  sleep_enable();
  sleep_mode();  // Goes to sleep
  // Wakes on interrupt
  sleep_disable();
}

// Wake on button press
attachInterrupt(digitalPinToInterrupt(2), wakeup, LOW);
```

Reduce Clock Speed

```
8MHz instead of 16MHz uses half power
Change fuses or use internal oscillator

Internal RC oscillator at 1MHz = very low power
```

Turn Off Peripherals

```cpp
// Disable ADC
ADCSRA &= ~(1 << ADEN);

// Disable analog comparator
ACSR |= (1 << ACD);
```

Brown-Out Detection

```cpp
// Disable BOD during sleep
sleep_bod_disable();
```

Power Consumption

Mode Current Battery Life (2000mAh)
Active 16MHz 15mA 5.5 days
Active 1MHz 2mA 41 days
Idle 1mA 83 days
Power-down 0.001mA 200+ years!

Strategy:

· Sleep 99% of time
· Wake briefly on button press
· Display, then sleep again
· Months of battery life
